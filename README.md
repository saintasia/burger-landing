# Burger Landing Page
Responsive one page burger joint landing page made with *SASS*, *HTML5*, *jQuery*　as well as *FontAwesome*, *Animate.css*, *Google Fonts* and *wow.js*. 
レスポンシブバーガーカフェのランディングです、*SASS*、 *HTML5*、と *jQuery*　で作成しております。
プラグインとフォントは*FontAwesome*、*Animate.css*、*Google Fonts* と *wow.js*

[View Live/ライブ表示](https://saintasia.github.io/burger-landing/)
